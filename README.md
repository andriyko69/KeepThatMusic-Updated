# Keep That Music (NeoForge 1.21.1)

This is a NeoForge 1.21.1 port of **Keep That Music**, originally developed for Fabric.

## Description

Keep That Music prevents Minecraft from unnecessarily stopping or replacing the currently playing background music during events such as:

- Dimension changes (e.g. Nether portals)
- Player respawn
- Reconnecting to a singleplayer world
- Pausing the game
- Returning to the title screen

Instead of interrupting the track, the music continues seamlessly.

This mod is fully client-side.

## Port Information

This version is a direct port of the Fabric 1.20.1 release to NeoForge 1.21.1.  
No gameplay changes were introduced beyond adapting the implementation to NeoForge and modern Minecraft internals.
